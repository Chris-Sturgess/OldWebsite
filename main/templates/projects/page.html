Projects

These are just a few of the recent projects I've been a part of.


Crosby - A tablet application designed using HTML5, Zepto.js, and Ruby on Rails to measure the reaction time of medical patients. Built to tablet devices with PhoneGap.

Foodlink - A mobile application for the Waterloo Region developed with HTML5, PhoneGap, and the Google Maps API. Designed to allow users to find local food stores.

Wheatley - A Playbook application designed using HTML5 designed as a sign-in system for companies. Keeps track of current guests, with many customizable fields for data. Also creates QR Code badges for guests, allowing them to scan to sign in instead of re-entering all other information.

FoodAtTheHub.com - An HTML website using Ruby on Rails to store information regarding food currently available at the Communitech Kitchen. Available at www.foodatthehub.com

Yarn Shop - A native iOS application that allows users to view and favourite all the yarn available at Camilla Valley Farm Weaver's Supply. Currently available for free in the App Store. More information available on the Yarn Shop App Store page.

www.christophersturgess.com - An online resume built using backbone.js and require.js for the main website, as well as Sencha Touch 2.0 for the mobile site. Available here.
